

Contensive Menu Styles

The FlyoutPanel is the overall window pane of buttons that becomes visible when the menu is clicked
The FlyoutButton the the button on the top that hosts the menu 'panel'



HTML structure

<DIV class="ccFlyoutPanel">
<A class="ccFlyoutPanelButton">Child Menu Button 1</A>
<A class="ccFlyoutPanelButton">Child Menu Button 2</A>
<A class="ccFlyoutPanelButton">Child Menu Button 3</A>
</DIV>




ccFlyoutPanel
	The background color of the panel
	Used in a <Div style=""> tag around each FlyoutPanelButton <A...> tag

	.ccFlyoutPanel { 
		BACKGROUND-COLOR: #e0e0e0; 	
		Z-INDEX: 101;  
		VISIBILITY: hidden;  
		POSITION: absolute;
		}	

ccFlyoutPanelFlyoutText
	When a Panel row contains a flyout menu, this style is on the text

	.ccFlyoutPanelFlyoutText { 		
		CURSOR: default;  
		} 

ccFlyoutPanelFlyoutPtr
	When a Panel row contains a flyout menu, this style is on the arrow

	.ccFlyoutPanelFlyoutPtr {
		MARGIN-RIGHT: -0.75em;
		text-decoration: none;
		} 

ccFlyoutPanelSep
	Horizontal rules between panel rows (not used)

	.ccFlyoutPanelSep { 			
		BORDER-TOP: #909090 1px solid;  
		MARGIN: 8px 2px;  
		BORDER-BOTTOM: #f0f0f0 1px solid 
		}

ccFlyoutButton
	The style in the <A tag around the top navigation buttons that open the menu
	Used around the Caption in an <A style=""> tag

	A.ccFlyoutButton { 
		color: #FFFFFF;
		text-decoration: none;
		FONT-WEIGHT: bold;  
		FONT-SIZE: 8pt;  
		FONT-STYLE: normal;  
		font-family: "MS Sans Serif", Arial, sans-serif; 
		BACKGROUND-COLOR: transparent;  
		TOP: 0px;  
		LEFT: 0px;  	
		PADDING-TOP: 2px;  
		PADDING-RIGHT: 6px;  
		PADDING-BOTTOM: 2px;  
		PADDING-LEFT: 6px;  
		Z-INDEX:100;  
		CURSOR: hand;
		}

	A.ccFlyoutButton:hover { 
		text-decoration: underline;
		}

ccFlyoutButtonDown
	The style added to the <A tag when the menu system is opened
	Used around the Caption in an <A style=""> tag

	A.ccFlyoutButtonDown {
		BACKGROUND-COLOR: #e0e0e0;  
		}



ccFlyoutPanelButton
	The buttons that are on Flyout Panels.
	Used around the Caption in an <A style=""> tag

	A.ccFlyoutPanelButton { 
		TEXT-DECORATION: none;   
		FONT-WEIGHT: bold;  
		FONT-SIZE: 8pt;  
		FONT-STYLE: normal;
		text-decoration: none;
		FONT-FAMILY: "MS Sans Serif", Arial, sans-serif; 
		PADDING-TOP: 3px;  
		PADDING-LEFT: 1em;  
		PADDING-RIGHT: 1em;  	
		PADDING-BOTTOM: 3px;  
		CURSOR: hand;  
		WHITE-SPACE: nowrap;  
		DISPLAY: block  
		}

	A.ccFlyoutPanelButton:link{
		text-decoration: none;
		CURSOR: hand;
		}

	A.ccFlyoutPanelButton:visited{
		text-decoration: none;
		CURSOR: hand; 
		}

	A.ccFlyoutPanelButton:hover{
		text-decoration: underline;
		CURSOR: hand;
		BACKGROUND-COLOR: #A0A0A3
		}



ccFlyoutPanelButtonDown
	When a panel button is selected, this style is added to the <A class="ccFlyoutPanelButton"> class in the panel.

	A.ccFlyoutPanelButtonDown {
		text-decoration: underline;
		BACKGROUND-COLOR: #A0A0A3;
		}
