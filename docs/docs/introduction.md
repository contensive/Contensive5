# What is Contensive?
Contensive is an open-source .NET library that creates a simple extensibility mechanism for online applications and websites on Amazon Web Services.

A developer creates snippets called add-ons that are deployed to the platform. Add-ons use a simple interface pattern and can be configured as end-points, background process tasks, website design blocks, etc.  

During execution add-ons are passed a dependency injection object that provides an API for resources like filesystem, cache, database, email , etc.

The architecture, design and development patterns are documented to facilitate AI development.
