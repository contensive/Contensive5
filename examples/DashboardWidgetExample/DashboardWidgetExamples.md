
## Examples
This is an example of pattern dashboard-widget-pattern in https://github.com/contensive/Contensive5/blob/master/patterns/dashboard-widget-pattern.md

### Number Widget
- display the number of users currently online.
- refresh the view periodically
- if a user clicks on the number they are forwarded to the Users online report
- dotnet code example https://github.com/contensive/Contensive5/blob/master/source/Processor/Addons/WidgetDashboardWidgets/UsersOnlineNumberWidget.cs
- installs in the aoBase50.xml collection, https://github.com/contensive/Contensive5/blob/master/source/Processor/aoBase51.xml

### Pie Chart Widget
- display a pie chart of the different types of membership
- options for top 2, top 6 and top 10 typeof memberships
- if the user clicks, it goes to the Contensive website
- dotnet code example https://github.com/contensive/Contensive5/blob/master/source/Processor/Addons/WidgetDashboardWidgets/SamplePieChartWidget.cs
- installs in the aoBase50.xml collection, https://github.com/contensive/Contensive5/blob/master/source/Processor/aoBase51.xml
